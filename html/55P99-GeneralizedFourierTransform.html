generalized Fourier transform


Definition 0.1.

Fourier-Stieltjes transform:

Given a positive definite, measurable function f⁢(x) on the interval (-∞,∞) there exists a monotone increasing, real-valued bounded function α⁢(t) such that:

f⁢(x)=∫ℝei⁢t⁢x⁢d⁢(α⁢(t)), (0.1)

for all x∈ℝ except a ‘small’ set, that is a finite set which contains only a small number of values. When f⁢(x) is defined as above and if α⁢(t) is nondecreasing and bounded then the measurable function defined by the above integral is called the Fourier-Stieltjes transform of α⁢(t), and it is continuous in addition to being positive definite.
