homotopy of maps


Let X,Y be topological spaces, A a closed subspace of X and f,g:X→Y continuous maps. A homotopy of maps is a continuous function F:X×[0,1]→Y satisfying

  1. 1.

    F⁢(x,0)=f⁢(x) for all x∈X

  2. 2.

    F⁢(x,1)=g⁢(x) for all x∈X

  3. 3.

    F⁢(x,t)=f⁢(x)=g⁢(x) for all x∈A,t∈[0,1].

We say that f is homotopic to g relative to A and denote this by f≃g r⁢e⁢l⁢A. If A=∅, this can be written f≃g. If g is the constant map (i.e. g⁢(x)=y for all x∈X), then we say that f is nullhomotopic.
